GET Health Check

Cek status API server

/api/health
GET Get All Manga

Dapatkan daftar semua manga dengan pagination

/api/manga?page=1&limit=20

Query Parameters:

page - Nomor halaman (default: 1)
limit - Jumlah per halaman (default: 20)
search - Cari berdasarkan judul
genre - Filter berdasarkan genre
GET Get Manga by Slug

Dapatkan detail manga berdasarkan slug

/api/manga/:slug

Example:

/api/manga/99-wooden-stick
/api/manga/eleceed
GET Get Manga Details (Full)

Dapatkan detail lengkap manga dengan metadata

/api/manga/:slug/details
GET Get Chapters

Dapatkan semua chapter untuk manga tertentu

/api/chapters/:slug

Query Parameters:

page - Nomor halaman (default: 1)
limit - Jumlah per halaman (default: 50)
GET Get Specific Chapter

Dapatkan detail chapter spesifik dengan gambar

/api/chapters/:slug/:chapterNumber

Example:

/api/chapters/99-wooden-stick/1
GET Get All Genres

Dapatkan daftar semua genre yang tersedia

/api/genres
GET Search Manga

Pencarian lanjutan dengan filter

/api/search?q=academy

Query Parameters:

q - Kata kunci pencarian
genre - Filter berdasarkan genre
page - Nomor halaman
limit - Jumlah per halaman
GET Latest Updates

Dapatkan manga yang baru diupdate

/api/latest-updates?limit=20
GET API Statistics

Dapatkan statistik API

/api/stats
